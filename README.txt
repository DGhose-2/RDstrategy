The file MAIN can be used to reproduce all results. Ensure that utility files

-  data_utils.py
-  nlp_utils.py
-  model_utils.py

are all in the same working directory as main.ipynb.


GoogleScholar.ipynb contains the code for extracting project outcome information from Google Scholar, which depends upon library `scholarly'.
'scholarly' is often obstructed by Google's reCAPTCHA, resulting in IP bans, especially if searching by title rather than by author.
